Anthropic Engineering: Scaling Managed Agents: Decoupling the brain from the hands

Anthropic 推出了 Managed Agents，这是一个在 Claude Platform 上运行的托管服务，旨在解决长期运行 Agent 的扩展问题。团队采用了类似操作系统的虚拟化思路，将 Agent 组件解耦为 session（日志）、harness（循环逻辑）和 sandbox（执行环境）三个接口，实现了“大脑”与“双手”的分离。这种设计允许各个部分的实现独立替换和演进，避免了将所有组件耦合在一个容器中导致的“宠物”式管理难题，从而提高了系统的稳定性和可维护性。详情

Claude Blog: New in Claude Managed Agents: self-hosted sandboxes and MCP tunnels

Claude Managed Agents 新增了自托管沙盒和 MCP 隧道功能，允许 Agent 在你控制的基础设施中运行工具，并连接到你私有网络中的 MCP 服务器。通过 self-hosted sandboxes，敏感文件和服务不会离开你的企业边界，你可以选择 Cloudflare、Daytona、Modal 或 Vercel 等提供商来处理计算和隔离。而 MCP tunnels 则通过一个轻量级网关，让 Agent 安全地访问内部数据库、私有 API 和知识库，无需暴露公网端点。详情
