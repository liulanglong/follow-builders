### Claude Blog: Claude Code now supports artifacts
Claude Code 新增 **artifacts** 功能，能将工作过程转化为可实时更新、可分享的可视化页面，包括 PR 演示、系统说明、仪表盘和发布清单等。artifacts 基于会话的完整上下文自动生成，无需额外搭建基础设施。页面在 Claude Code 更新时会原地刷新，团队成员看到的始终是最新版本，并支持版本历史回溯。内部测试中最常见的场景是调试：工程师在站会前启动事故调查，Claude Code 分析日志后发布一个包含时间线、可疑提交和错误率图表的 artifact，随着调查推进自动更新。所有 artifact 默认对作者私有，仅限组织内认证成员查看，管理员可通过合规 API 管理访问权限和留存策略。[详情](https://claude.com/blog/artifacts-in-claude-code)
